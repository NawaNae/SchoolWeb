指令介紹 換行代表一個新的指令 指令參數用空格隔開，用空格請用全形
  1. background 圖片網址
    • 指定背景為該網址圖片
  2. character 圖片網址 角色名稱 角色id(不可與其他重複)(可不填)
    • 指定角色為該網址圖片
    • 顯示的角色名稱複
    • id不可與其他角色重複
  3. talk 對話框文字
    • 改變對話框內文字
  4. text 全螢幕文字
    • 改變全螢幕文字內容
    • 預設不顯示，請先使用show text
  5. tag 標籤名稱
    • 在時間軸中插入一個標籤，以利選擇或跳躍
    • 請不要與其他名稱重複
  6. jump 標籤名稱
    • 跳至某個標籤的位置開始執行
    • 若找不到標籤執行下一步
    • 若標籤名稱為NEXT_STEP則執行下一步
  7. select 選項文字1,標籤名稱1;選項文字2,標籤名稱2; .... ;選項名稱n,標籤名稱n
    • 逗號前方為選項文字內容，後方為欲跳往的標籤名稱
    • 用分號分隔所有選項
    • 若找不到標籤執行下一步
    • 若標籤名稱為NEXT_STEP則執行下一步
  8. show 物件
    • 顯示某個物件
    • 物件列表
      1. character
        • 角色圖
      2. background
        • 背景圖
      3. talk
        • 對話框
      4. text
        • 全螢幕文字框
  9. hide 物件
    • 隱藏某個物件
    • 物件列表
      1. character
        • 角色圖
      2. background
        • 背景圖
      3. talk
        • 對話框
      4. text
        • 全螢幕文字框
